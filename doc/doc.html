Script python

Le script python "hashtable.py" permet de pouvoir générer une hashtable dans un fichier en format json. Il y a plusieurs méthodes pour utiliser notre script :

  • La première, en exécutant simplement le script. Dans le code, il y a des valeurs par défaut pour le nom du dossier d'export, ainsi que les chemins d'accès aux fichiers texte.
  • pour la seconde méthode, il est possible de spécifier plusieurs paramètres d'entrée. un exemple de commande dans le cmd :
    py hashtable.py hash_folder french;dict/french.txt english;dict/english.txt
    Ici, le script va générer deux fichiers json (un par rapport au fichier "french.txt" et un autre pour le "english.txt") dans un dossier nommé hash_folder.

Attention ! Le script va d'abord supprimer un probable dossier qui aurait le même nom, vérifiez donc à utiliser un nom de dossier qui n'existe pas encore, ou que le dossier déjà existant n'est plus utile.

Application de détection

Au niveau de l'application de détection, son utilisation est très simple.

Tout d'abord nous avons un champs pour entrer du texte. Il suffit d'entrer une phrase de plusieurs mots, pour que notre algorithme détecte la langue dans laquelle la phrase est écrite. Une fois la langue détectée, un petit drapeau représentant la langue sera affiché en dessus.

En dessous du drapeau, il est également affiché le pourcentage que le texte écrit soit bien dans la langue détectée.

Au niveau de la correction, le texte entré est copié en dessous, et les erreurs (par rapport à la langue détectée en cours) sont surligné en rouge.

A quoi sert la liste déroulante en bas ? En sélectionnant une langue, des informations pour le dictionnaire de mots correspondant seront affichées. Ces données sont :

  • Number of words : correspond au nombre de mots présents dans le dictionnaire
  • Average time access : le temps d'accès moyen à un mot dans la table
  • Number of collisions : le nombre de collisions présentes dans la table de hachage
  • Filling rate of the HashTable : le taux de remplissage de notre table de hachage
